# kvazarcomp-site

Modern static rebuild of [kvazarcomp.ru](https://kvazarcomp.ru) on Astro + TypeScript + TailwindCSS.

## Commands

```
npm install        # install dependencies
npm run dev        # start the Astro dev server
npm run build      # astro check && astro build (outputs dist/)
npm run preview    # preview the built site
npm run check      # astro check (type + diagnostics)
npm run scrape     # scrape the legacy site into src/content/ (added in FEAT-002)
npm run smoke      # post-build smoke test (added in FEAT-004)
```

Full documentation (deploy, scrape options, content editing) will be added in FEAT-004.
